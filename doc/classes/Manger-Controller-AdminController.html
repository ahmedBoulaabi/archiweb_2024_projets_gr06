public getAllRecipes() : void

Retrieves all the recipes from the database and returns them in a JSON format. If recipes are found, the function dynamically includes a PHP view file (for example, a table of recipes), captures its output using output buffering, and returns this HTML as part of the JSON response. If no recipes are found, a JSON response with a message indicating the absence of recipes is returned.

Return values
void —

Outputs a JSON-encoded string. On success, this string includes HTML content generated by including a view file that displays the recipes in a user-friendly format. On failure, it includes a simple message indicating that no recipes are available.
